== Description ==

**Store Customizer For Woocommerce** using for editing the WooCommerce store and product pages, cart and checkout pages and also your user account page and Thank you page.

**Store Customizer For Woocommerce** DOES NOT override the WooCommerce templates, the plugin simply adds extra design options and advanced shop functionality to your existing WooCommerce pages which are designed by the theme you are using.

A simple and easy way to Customize woocommerce checkout page, disable unwanted checkout feelds, free checkout .

**Store Customizer For Woocommerce** customized my account page to edit and remove unwanted tab
